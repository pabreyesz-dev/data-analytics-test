## Looker Modeling Language / LookML Semantic Layer

<Español>

Este proyecto incluye una capa semántica completa en Looker Modeling Language construida en una base de datos PostgreSQL
para el análisis avanzado de datos en un comercio electrónico simulado.

Características clave:

- Dimensiones y métricas gobernadas
- Capa semántica de exploración para empresa
- Correcta unión de las cardinalidades
- Definiciones reutilizables de ganancias, beneficios y KPIs
- Compatible con dashboards ejecutivos y automatización de inteligencia de negocios

La capa semántica garantiza la coherencia entre dashboards y evita duplicidad en las métricas,
todo esto siguiendo las mejores prácticas en el análisis de datos.



<English>

This project includes a full LookML semantic layer built on top of a PostgreSQL
e-commerce analytics database.

Key features:

- Governed dimensions and metrics
- Business-friendly Explore layer
- Correct join cardinalities
- Reusable revenue, profit, and KPI definitions
- Ready for executive dashboards and self-service BI

The LookML layer ensures consistency across dashboards and prevents metric duplication,
following analytics engineering best practices.
