# SQL Analytics Layer

<ESPAÑOL>
Esta carpeta contiene consultas SQL avanzadas orientadas a generar métricas y KPIs de negocio a partir del modelo relacional.

---

## Enfoque

Las consultas están diseñadas para:
- Analítica descriptiva
- Agregaciones complejas
- Métricas reutilizables
- Preparación para capas BI y dashboards

---

## Técnicas Utilizadas

- Common Table Expressions (CTEs)
- JOINs entre múltiples tablas
- Agregaciones (SUM, COUNT, AVG)
- Métricas por dimensión temporal y categórica

---

## KPIs Generados

- Ingresos por período
- Órdenes por cliente y categoría
- AOV (Average Order Value)
- Performance por producto

---

## Uso

Estas consultas pueden ser utilizadas:
- Directamente en PostgreSQL
- Como base para vistas analíticas
- Como capa previa a BigQuery o Looker


<ENGLISH>
This folder contains advanced SQL queries designed to generate business metrics and KPIs from the relational model.

---

## Focus

The queries are designed for:
- Descriptive analytics
- Complex aggregations
- Reusable metrics
- Preparation for BI layers and dashboards

---

## Techniques Used

- Common Table Expressions (CTEs)
- JOINs across multiple tables
- Aggregations (SUM, COUNT, AVG)
- Metrics by time dimension and categorical dimension

---

## KPIs Generated

- Revenue by period
- Orders by customer and category
- AOV (Average Order Value)
- Performance by product

---

## Uses

These queries can be used:
- Directly in PostgreSQL
- As a basis for analytical views
- As a pre-layer for BigQuery or Looker
