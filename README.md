# E-commerce Analytics Platform – BI & Data Analytics

<ESPAÑOL>
Proyecto de analítica de datos end-to-end orientado a inteligencia de negocios (BI), que cubre desde el diseño del modelo relacional hasta la visualización de KPIs estratégicos utilizando herramientas líderes de la industria.

El objetivo del proyecto es simular un entorno real de e-commerce y demostrar competencias en SQL avanzado, modelado de datos, herramientas BI y visualización ejecutiva.

---

## Objetivos del Proyecto

- Diseñar un modelo de datos relacional orientado a analítica
- Implementar consultas SQL avanzadas para generación de KPIs
- Construir dashboards ejecutivos en Power BI y Tableau
- Explorar una arquitectura cloud-ready compatible con BigQuery y Looker
- Presentar insights claros para toma de decisiones de negocio

---

## Arquitectura General

**Fuente de datos**
- PostgreSQL (modelo transaccional normalizado)
- Datos poblados mediante scripts de seed realistas

**Capa analítica**
- SQL avanzado (CTEs, agregaciones, joins)
- Vistas analíticas reutilizables
- Preparación para BigQuery / Looker

**Visualización**
- Power BI (dashboard principal)
- Tableau (dashboard alternativo)
- Enfoque en KPIs ejecutivos

---

## KPIs Analizados

- Ingresos Totales
- Órdenes Totales
- Average Order Value (AOV)
- Evolución de ingresos en el tiempo
- Órdenes e ingresos por categoría de producto

---

## Tecnologías Utilizadas

- **Base de datos:** PostgreSQL
- **Lenguaje SQL:** SQL avanzado
- **BI & Visualización:** Power BI, Tableau
- **Cloud & Analytics:** BigQuery (exploratorio)
- **Control de versiones:** Git / GitHub
- **Herramientas:** VS Code, pgAdmin
